# Animal genetic testing

### Приложение, которое позволяет сохранять данные генетических тестов животных, а также выполнять расчет агрегированной статистики.
Добавлять, редактировать и удалять данные могут лишь айтентифицированные пользователи. 
Аутентификация реализована с использованием access и refresh токенов.  
Статус: в разработке.

# Стек технологий

- Python 3.14
- FastAPI — веб-фреймворк.
- SQLAlchemy — ORM для работы с базой данных.
- Pydantic — валидация данных.
- Alembic — миграции базы данных.
- PostgreSQL — база данных.
- Uvicorn — ASGI-сервер.
- Pytest — тестирование.

# Установка и запуск

1. Установка uv.
   Если у вас еще не установлен uv, установите его одной командой:

```bash
# macOS/Linux
curl -LsSf astral.sh | sh
# Windows
powershell -c "ir astral.sh | iex"
```

2. Клонирование и настройка проекта.

```bash
git clone https://github.com/aleksandrgurov2605/animal-genetic-testing.git
cd animal-genetic-testing
```

3. Создание окружения и установка зависимостей.

```bash
uv sync
```

4. Настройка переменных окружения.  
   Переименуйте файл "example.env ", изменив название на ".env" и задайте значения переменных.


5. Запуск приложения.  
Создайте базу данных PostgreSQL.  
После этого примените миграции, выполнив команду:
```bash
uv run alembic upgrade head
```

Запустите приложение

```bash
uv run uvicorn app.main:app --reload
```


### Приложение будет доступно по адресу: http://127.0.0.1:8000
